# Contributing

Things aren't very well organized as we work towards v1. The main things we need help with are:

- Testing
- Feedback
- UI/UX
- [Sponsors](https://github.com/sponsors/wceolin)

If you have any feedback, please [start a discussion](https://github.com/orgs/zoonk/discussions).

## Contribution areas

### For developers

- Look for [good first issue](https://github.com/zoonk/zoonk/labels/good%20first%20issue) or [help needed](https://github.com/zoonk/zoonk/labels/help%20needed) issues.
