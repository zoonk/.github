# Zoonk Roadmap

This is an overview of the main features and products we have planned. We'll work on all of these things but we can't promise when it's be available. For a detailed roadmap of the things we're currently working on, keep an eye on [projects page](https://github.com/orgs/zoonk/projects?query=is%3Aopen). All progress will be updated there.

## MVP/launch

- [x] Email/password authentication
- [x] Manage students/teachers
- [x] Guest users
- [x] Course editor
- [ ] Course collection
- [x] Format: Multiple choice
- [x] Format: Read-only
- [x] Format: Image
- [x] Format: Open-ended question
- [ ] Format: YouTube embed

## Products

- [x] Zoonk: Platform/white-label for creating interactive courses.
- [ ] Zoonk Learn: Learn anything using AI.
- [ ] Zoonk School: Learn science with practical examples of everyday life.
- [ ] Zoonk Courses: Marketplace for content creators.
- [ ] Zoonk Devs: API/elements for building customized learning apps.
- [ ] Zoonk Play: Immersive game where players can learn anything by solving practical missions in the real world.
