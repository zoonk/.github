# About hiring, partnerships, and contributing

[UneeBee](https://github.com/zoonk/uneebee) caught some attention in the past few days and I've been getting a few questions about partnerships, hiring, getting involved, etc. I really appreciate the interest but there are a few things to keep in mind.

## Hiring

At the moment, I'm working solo on this, trying to bootstrap it as much as I can. So, I don't have any budget for hiring.

## Contributing

The project is completely open-source and transparent, so all contributions are welcome. I'm planning to do all communication and work on GitHub. Feel free to check out the org's profile, especially the [how we work section](https://github.com/zoonk#-how-we-work).

Basically, all information about the project should be there or here (GitHub Discussions). If you didn't find an answer for your question, ask it here.

There are lots of moving parts and things aren't very well organized yet as I work towards v1, though. You can see the [roadmap here on GitHub too](https://github.com/orgs/zoonk/projects/11). So, I understand that isn't the best contributing experience so far. Things will get better over time.

If I do eventually hire people to help out, I'll definitely favor those who contributed to the open-source project. However, keep in mind that contributing isn't a guarantee that you will eventually get hired because, as I said, this is a small project for now and my budget is really short. I'm not even paying myself at the moment. 😅

Early interest is promising so far but I have no idea if I'll succeed in turning this project into an actual business or not.

## Partnerships

Partnerships are also welcome. You can suggest a partnership here on GitHub Discussions (preferred) or send an email to will@zoonk.org explaining what you have in mind and what's the expected outcome.

## Questions

Feel free to add a comment below or [start a new discussion](https://github.com/orgs/zoonk/discussions/new/choose).
