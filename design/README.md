# Design

## Philosophy

- Clean
- Minimalist
- Inspired by [Apple](https://developer.apple.com/design/) and the [Nielsen Norman Group](https://www.nngroup.com/)

> [!IMPORTANT]
> We should always follow [Apple's Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/). Our design should make people feel **inspired**, **joy**, **satisfaction**, **excitement**.
